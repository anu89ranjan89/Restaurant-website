**Spice Delight Restaurant Website**

Spice Delight is a responsive restaurant website built using **HTML, CSS, and JavaScript**.  
The project initially started as a static website using HTML and CSS. After learning JavaScript, I revisited the project and enhanced it with dynamic features and interactive functionality.

## 🚀 Features

- 🔍 **Search Functionality** – Users can search menu items dynamically.
- 📊 **Price Sorting** – Sort menu items from Low → High and High → Low.
- 🛒 **Dynamic Cart System**
- 🔢 **Cart Quantity Tracking**
- 🧾 **Real-time Total Price Calculation**
- 🧭 **Smooth Navigation**
- 📱 **Responsive Design**

## 🧠 Concepts Used

This project also demonstrates the use of **basic Data Structures & Algorithms concepts** within a front-end application:

- **Linear Search** → used for menu item search functionality
- **Sorting Algorithm** → used for sorting menu items by price
- **HashMap Concept (JavaScript Object)** → used to store cart items and quantities
- **DOM Manipulation** → dynamically updating cart items, totals, and UI elements
- **Array Traversal** → iterating through menu cards for search and sorting

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
